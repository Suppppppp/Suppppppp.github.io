# 1번

Option 1:
```swift
message = "Good morning, Dave!"
```

Option 2:

```swift
var operatingSystem = "macOS"
```



<details>
<summary> 💡 정답 </summary>
<div markdown="1">
<br>
정답은 2번입니다.<br>
option1은 새 변수를 만들기 위해서는 `var`와 함께 시작해야합니다 `var` message = "Good morning, Dave!"
<br>
option2는 정답입니다. operatingSystem이라고 불리는 변수가 생성됩니다.
</div>
</details>
